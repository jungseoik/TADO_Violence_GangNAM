# VLM Instruction Tuning Project

## 📝 프로젝트 소개

이 프로젝트는 Vision-Language Model (VLM)의 instruction 튜닝을 위한 파이프라인을 제공합니다. 데이터 전처리, 자동 라벨링, 모델 학습 및 평가의 전 과정을 체계적으로 관리하고 실행할 수 있도록 설계되었습니다.

- [프로젝트 구조](docs/project_structure.md)
